// anti-slop

Slop is the gap between what a senior engineer writes at 2 AM and what an LLM produces. It compiles, it passes tests, but it's unnecessary weight.

  • Code smells вҖ” wrapper types, single-case enums, defensive guards, docstrings on 3-line helpers
  • Text smells вҖ” "delve into", "leverage", meta-commentary, hedge language
  • Design smells вҖ” AI startup gradients, glassmorphism, "Empower your business" headlines

// token-saver

Claude Code burns tokens on verbose output. This hook system compresses it automatically with 32 regex rules.

  • Flag rules вҖ” git log becomes git log --oneline -20
  • Pipe rules вҖ” test output compressed to failures + summary
  • 10 filters вҖ” pytest, cargo, Go, lint, tsc, Docker, k8s, generic
  • Fail-safe вҖ” errors exit silently, sessions never break

// spectral-agents

12 agents that analyze your code through specialized lenses. Run one at a time or ask Claude to dispatch multiple in parallel.

  • Security audit, architecture, performance
  • Code quality, UX review, accessibility (WCAG 2.1 AA)
  • API design, database review, data integrity
  • Planner, debugger, pre-ship pipeline

// more-skills

Every skill installs as a symlink вҖ” lightweight, always up to date.

  • UX Review вҖ” structured user experience audit
  • Ad Review вҖ” ad copy and creative analysis
  • Product Diagnostics вҖ” validate product decisions
  • Multi-Stack Scaffolder вҖ” research-driven project setup
  • CLAUDE.md Generator вҖ” project onboarding files
