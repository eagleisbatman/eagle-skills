# Eagle Clean Sheet

Modern Excel spreadsheet generation with consistent styling.

## Command

```
/eagle-clean-sheet
```

Also triggers on: "spreadsheet", "excel", "xlsx", "create a spreadsheet", "export to Excel"

## Design system

- Same color tokens as Clean Doc (monochrome)
- **Body:** Helvetica
- **Technical columns:** Courier New (event names, endpoints, identifiers)
- Gray header rows with frozen panes
- Thin gray borders
- Alternating row fills for large datasets
- Auto-sized columns

## Usage

Use standalone for any spreadsheet task, or as an export format from Product Diagnostics (scorecards, funnels, findings as sortable/filterable data). Say "excel" or "spreadsheet" when running Product Diagnostics.
