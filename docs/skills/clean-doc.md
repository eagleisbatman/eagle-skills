# Eagle Clean Doc

Modern Word document generation with a clean, monochrome design system.

## Command

```
/eagle-clean-doc
```

Also triggers on: "word doc", "docx", "create a document", "write a report", "generate a Word file"

## Design system

- **Body:** Helvetica 10.5pt
- **Code:** Courier New 9.5pt
- **Palette:** Black, gray, white only
- **Margins:** 0.4" top/bottom, 0.5" sides
- **Tables:** Gray headers, clean borders
- **Code spans:** Subtle gray background

## Usage

Use standalone to generate any Word document, or as the output format for review skills (UX Review, Product Diagnostics, Ad Review). Say "word" or "docx" when running a review skill to get this format.
