// anti-slop

Slop is the gap between what a senior engineer writes at 2 AM and what an LLM produces. It compiles, it passes tests, but it's unnecessary weight.

  • Code smells ‚ÄĒ wrapper types, single-case enums, defensive guards, docstrings on 3-line helpers
  • Text smells ‚ÄĒ "delve into", "leverage", meta-commentary, hedge language
  • Design smells ‚ÄĒ AI startup gradients, glassmorphism, "Empower your business" headlines

// token-saver

Claude Code burns tokens on verbose output. This hook system compresses it automatically ‚ÄĒ and learns your workflow over time.

  • Flag rules ‚ÄĒ git log becomes git log --oneline -20
  • Pipe rules ‚ÄĒ test output compressed to failures + summary
  • Auto-observer ‚ÄĒ logs verbose commands, promotes them to rules
  • Fail-safe ‚ÄĒ errors exit silently, sessions never break
